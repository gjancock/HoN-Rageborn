# Rageborn
Any thoughts on Rageborn?

How to Use
-------------------------------------
Game Settings

1. Set the game resolution to 1024×768.
2. Log out of any currently logged-in account, then uncheck “Remember Me”.
3. Ensure the Teleport Stone hotkey is set to ALT + T.
4. Make sure the minimap is in its default position (bottom-left corner of the screen).
5. Ensure “Minimap Right Click” is enabled in the game settings.

Windows Settings

1. Set the Windows display resolution to 1920×1080.
2. Make sure a “Heroes of Newerth Reborn” desktop shortcut exists and is visible when running the script.
3. Minimize all other applications after starting the script to avoid interruptions (this will be improved in future updates).
4. Do not move the mouse cursor or click anywhere while the script is running.

Notes
-------------------------------------
All generated account usernames will be saved in a text file named
signup_users.txt, located in the same folder as Rageborn.exe.
